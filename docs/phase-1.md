# Phase 1 — 대회용 구현

## 목표

서비스 핵심 루프(기록 → 분석 → 출력)를 완성하여 대회 시연이 가능한 상태를 만든다.

## 기능 명세

### 0. 랜딩 (`app/landing/`)

- 서비스 가치 제안 히어로 섹션
- Free / Pro 플랜 비교 테이블
- CTA → 회원가입 유도

### 1. 계정·설정 (`app/(auth)/`)

- 회원가입: 이메일 + 소셜(Google, Kakao)
- 로그인 / 로그아웃
- 마이페이지: 프로필 편집, 플랜 확인

### 2. 커리어 아카이브 (`app/(main)/archive/`)

- 유형별 직접 입력 (프로젝트, 대외활동, 인턴, 수상, 자격증 등)
- 커스텀 템플릿: 유저가 직접 입력 필드를 정의
- 각 경험에 태그·날짜·요약 첨부

### 3. 커리어 분석 (`app/(main)/analysis/`)

- 기록 리스트 + 필터 (유형, 기간, 태그)
- 키워드 자동 분석·태깅 (빈도 기반)
- 시각화 리포트: 역량 분포 차트, 타임라인 뷰

### 5. 익스포트 (`app/(main)/export/`)

- 스마트 이력서 자동 생성 (기록 기반 → PDF/웹)
- 자소서 초안 작성 (선택한 경험 조합 → 텍스트 생성)
- Free 플랜: 워터마크 포함 / Pro 플랜: 워터마크 제거

### 6. 커리어 대시보드 (`app/(main)/dashboard/`)

- 역량 가시화 리포트 (분석 결과 요약)
- 아이덴티티 한줄 요약 (AI 생성)
- 퀵 레코딩: 대시보드에서 바로 짧은 경험 메모 입력

## 우선순위

1. 아카이브(2) — 데이터가 없으면 나머지가 동작하지 않음
2. 분석(3) — 기록 위에 가치를 쌓는 핵심 기능
3. 익스포트(5) — 시연 시 “결과물”로 보여줄 수 있는 기능
4. 대시보드(6) — 진입점, 전체 요약
5. 계정(1) — 인증은 MVP 수준으로 먼저 구현
6. 랜딩(0) — 시연 직전 마무리
