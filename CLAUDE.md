# ARC — 프로젝트 컨텍스트

## 서비스 개요

ARC는 **포트폴리오 아카이빙 및 AI 연계 자동화 플랫폼**이다.
이름은 Story Arc에서 착안했으며, 한 사람의 경력과 경험 조각들을 모아 하나의 내러티브로 만들어주는 서비스다. Archive의 약어이고 방주(Ark)를 뜻하기도 한다.

## 핵심 철학

- 이력은 단순한 스펙 목록이 아니다. **정성적 경험의 총체**다.
- 링커리어, LinkedIn 등 타 플랫폼과 달리 정성적 데이터 수집·연결에 집중한다.
- **기록의 허들을 낮추는 것이 최우선**. UX는 항상 입력 편의성을 기준으로 판단한다.
- 주요 타겟은 **진로가 확정되지 않은 대학생**. 안정감을 주는 플랫폼이어야 한다.

## 개발 시 주의사항

- 컴포넌트 작성 시 **적응형 웹** 기준으로 Tailwind 클래스 작성
- 페이지 간 상태는 최대한 URL(searchParams)로 관리, 전역 상태 최소화
- AI 기능은 **백엔드 API 엔드포인트**로 분리. 프론트는 fetch/axios로 호출만 한다
- 워터마크 로직은 Pro 여부를 서버에서 검증 후 적용
