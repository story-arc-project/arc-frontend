# ARC

당신은 지금까지 몇 개의 활동을 했나요?

공모전, 동아리, 인턴십, 수업, 봉사활동 — 아마 꽤 많을 거예요.
그런데 그 경험들을 **왜 했는지**, **무엇을 느꼈는지**, 지금도 말할 수 있나요?

대부분은 이력서를 쓸 때가 되어서야 기억을 더듬습니다.
그리고 결국 비슷비슷한 문장으로 채워진 한 장짜리 문서가 완성되죠.

---

**ARC는 그 간극을 채우려고 만들어졌습니다.**

경험을 쌓는 순간부터 조금씩 기록해두면,
나중에 AI가 그 조각들을 연결해 하나의 이야기로 만들어줍니다.
이력서가 아니라, 당신이라는 사람의 **서사**로.

이름 ARC는 *Story Arc*에서 왔습니다.
당신의 경험에 숨어있는 흐름을 찾아내는 것 — 그게 ARC가 하는 일입니다.

---

## 무엇을 할 수 있나요

- **아카이브** — 활동 유형에 맞는 템플릿으로 경험을 기록하거나, AI와 대화하면서 자연스럽게 정리
- **분석** — 쌓인 기록에서 키워드와 패턴을 추출, 내가 어떤 사람인지 시각화
- **전략** — 진로 로드맵 설계, 내러티브 구축
- **익스포트** — 이력서·자소서 자동 초안 생성

---

## 기술 스택

- **Next.js 16** (App Router) + **TypeScript**
- **Tailwind CSS v4**
- **FastAPI** 백엔드와 REST 연동
- **Vercel** 배포

---

## 로컬 실행

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000)에서 확인할 수 있습니다.

```bash
npm run build       # 프로덕션 빌드
npm run lint        # ESLint
npm run typecheck   # TypeScript 타입 체크 (tsc --noEmit)
```
